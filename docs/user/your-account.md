# Your account

## Signing in {: #signing-in }

If the site asks for a username and password, enter the credentials you were given. Your session lasts about **seven days**; after that you may need to sign in again.

Use a **private or trusted device**. Do not share your password.

If the site uses **HTTPS** (padlock in the browser), your session is protected in transit. If you only see **HTTP**, avoid using this dashboard on public Wi‑Fi.

## Changing your password {: #change-password }

Open **Password** (or **Change password**) in the menu. You must enter your **current** password, then your **new** password twice. Pick something long and unique.

## Admin users {: #admin-users }

From **Users**, people with access can add more sign-ins or remove old ones. You cannot delete your own account while you are logged in as that user, and the last remaining user cannot be removed (so you are never locked out with zero accounts).

## Help without signing in {: #help-without-login }

The **Help** link works from the sign-in page too, so you can read these guides before you log in.
